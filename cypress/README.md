# Cypress Tests

This project uses [Cypress](https://www.cypress.io/) for end-to-end testing. Follow these steps to set up and run the tests.

## Installation

1. **Install Dependencies**

   ```bash
   npm install
   ```

2. **Install Cypress**

   ```bash
   npm install cypress --save-dev
   ```

3. **Open Cypress Runner**

   ```bash
   npx cypress open
   ```
